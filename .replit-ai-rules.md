# PLAN 57 — Agent Rules

You must not scan the whole repo.
You must not inspect node_modules.
You must not inspect .git.
You must not inspect package-lock.json unless asked.
You must work one file at a time.
You must use shell commands for search.
You must make minimal patches.
You must run the smallest possible test after each patch.
You must not run npm audit fix --force.
You must not reinstall dependencies unless package.json changes.
You must not optimize memory by reading every file.
